# Journal Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a public journal (blog-style) at `www.baltoratora.my/journal` — D1-backed, public read, secret-key write, deployed as part of the `baltoratora-landing` Pages project.

**Architecture:** Pages Function in `landing/functions/api/journal.ts` (auto-discovered by `wrangler pages deploy`), a self-contained static `landing/journal.html` page (same dark-cosmic theme as the landing page), and a journal button added to `landing/index.html`. Shares the existing `baltoratora-scores` D1 database. D1 binding and `JOURNAL_SECRET` env var must be configured manually in the Cloudflare dashboard (one-time, not in code).

**Tech Stack:** Cloudflare Pages Functions (TypeScript), D1 (SQLite), vanilla JS (no framework), HTML/CSS

## Task 1: Pages Function

- [ ] **Step 1: Create `landing/functions/api/journal.ts`**

```typescript
interface Env {
  DB: D1Database;
  JOURNAL_SECRET: string;
}

interface JournalEntry {
  id: number;
  title: string;
  body: string;
  created_at: string;
}

async function ensureTable(db: D1Database): Promise<void> {
  await db.exec(
    `CREATE TABLE IF NOT EXISTS journal_entries (
      id         INTEGER PRIMARY KEY AUTOINCREMENT,
      title      TEXT NOT NULL,
      body       TEXT NOT NULL,
      created_at TEXT NOT NULL
    )`,
  );
}

export const onRequestGet: PagesFunction<Env> = async ({ env }) => {
  try {
    await ensureTable(env.DB);
    const { results } = await env.DB.prepare(
      "SELECT id, title, body, created_at FROM journal_entries ORDER BY id DESC LIMIT 100",
    ).all<JournalEntry>();
    return Response.json({ entries: results });
  } catch (e) {
    return Response.json({ error: String(e) }, { status: 500 });
  }
};

export const onRequestPost: PagesFunction<Env> = async ({ request, env }) => {
  let data: Record<string, unknown>;
  try {
    data = (await request.json()) as Record<string, unknown>;
  } catch {
    return Response.json({ error: "invalid json" }, { status: 400 });
  }

  const { title, body, key } = data;
  if (typeof title !== "string" || typeof body !== "string" || typeof key !== "string") {
    return Response.json({ error: "missing fields" }, { status: 400 });
  }
  if (!title.trim() || !body.trim()) {
    return Response.json({ error: "empty fields" }, { status: 400 });
  }
  if (title.length > 200 || body.length > 10000) {
    return Response.json({ error: "too long" }, { status: 400 });
  }
  if (key !== env.JOURNAL_SECRET) {
    return Response.json({ error: "wrong key" }, { status: 401 });
  }

  try {
    await ensureTable(env.DB);
    const now = new Date().toISOString();
    const entry = await env.DB.prepare(
      `INSERT INTO journal_entries (title, body, created_at)
       VALUES (?, ?, ?) RETURNING id, title, body, created_at`,
    )
      .bind(title.trim(), body.trim(), now)
      .first<JournalEntry>();
    return Response.json({ entry }, { status: 201 });
  } catch (e) {
    return Response.json({ error: String(e) }, { status: 500 });
  }
};
```
